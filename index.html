TAREAS INICIALES

Tutorial

Instalacion de NodeJS

Para instalar nodeJS solo es necesario ir al la página web oficial de NodeJS, buscar la version de Node que mas te convenga y simplemente descargas instalador y seguir los pasos. Link de la pagina oficial de NodeJS:https://nodejs.org/en/

Para usar el NodeJS simplemente vas a tu respectiva terminal y escribes Node.

Instalacion de Express

Para instalar Express solo hace falta ir a una terminal y ejecutar el siguiente comando: npm install express --save

Instalacion de Atom

Para instalar Atom solo es necesario ir al la pagina web oficial de Atom, buscar la version de Atom que mas te convenga y simplemente descargas instalador y seguir los pasos. Link de la pagina oficial de Atom:https://atom.io/

Instalacion de Pandoc

Para instalar Pandoc hay que ir al sitio oficial de Pandoc y buscar la versión que mas que te convenga. Luego ir al enlace de github con los ficheros a descargar. Una vez ahí elige el que mas la version de windows y la descargas y sigue los pasos de instalacion. Link de la pagina oficial de Pandoc:http://pandoc.org/installing.html

Para usar Pandoc simplemente ahi que ir a la terminal y ejecutar el siguiente comando:

pandoc "nombredelarchivo" -o index.html"

Instalacion de GitHub Desktop

Para instalar GitHub Desktop solo es necesario ir al la pagina web oficial de GitHub Desktop buscar la version de la app que mas te convenga y simplemente descargas instalador y seguir los pasos. Link de la pagina oficial de GitHub Desktop:https://desktop.github.com/

Realizacion de la Practica

Tras instalar todos abrimos un fichero en Atom y escribimos en Markdown el tuorial. Luego mediante pandoc pasamos el fichero de Markdown a html. Lo subimos a GitHub y lo guardamos en la rama gh-pages para que se guarda como pagina web
